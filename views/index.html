Okta Event Hook: Sync user profiles from Okta Cloud to your database

Complete overview and step-by-step guide available on the Okta Developer Blog

A Node.js application written to receive Event Hooks from Okta to sync new user profiles to a database.

Verify and Post Event Hook Content: /UserTransfer
View all users in a database: /GetUsers
Wipe User DB with Post: /ClearUsers

Made with Glitch!
